# Plant Click Game

A simple Python game where you click on a plant to make it move to a random position.

## Requirements

To run this game, you need:

1. **Python 3.x** (recommended 3.10+)
2. **Tkinter** (usually included with Python)
3. **Pillow** library for image handling

You can install Pillow using pip:

```bash
pip install pillow
